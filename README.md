**💼 2026, гайды под заказ.** пишу персональные мануалы для хостеров и индивидуально:

```
├─ настройка vpn-стека (Xray/Reality, WireGuard, AmneziaWG, sing-box)
├─ протоколы, обфускация и обход dpi под конкретного исп
├─ серверные утилиты: мониторинг, fail2ban, systemd, ufw, bbr-тюнинг
├─ деплой с пустого vps до прод-конфига
└─ документация в стиле клиента: step-by-step, без воды, воспроизводимо
```

> не копипаст из интернета, а рабочие мануалы под конкретный стек, исп и задачу. nda/payout по договорённости, tg/mail для связи.

**💼 2025, свой vpn-сервис.** ~2 месяца жизни, в одно лицо:

```
├─ инфраструктура на 3 серверах, деплой и настройка с нуля
├─ tg-бот с авто-продажей подписок + интеграция с юкассой
├─ реклама, саппорт, эксплуатация на себе
├─ ~1500 платящих подписчиков за ~8 недель
└─ закрыл сам из-за рф-законов, проект продал
```

> опыт оставил понимание как устроен стек от рекламы до роутинга: биллинг, автоматизация, нагрузка, саппорт реальных людей, трафик под dpi.
